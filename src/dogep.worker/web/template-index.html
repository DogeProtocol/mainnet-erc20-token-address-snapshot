Doge Protocol is a community run project. There are no guarantees of any kind or value.
  1. 1) Number of DP coins is based on the formula NumberOfDogePTokens * 0.15, where NumberOfDogePTokens is DogeP tokens in wallet address as of Ethereum snapshot block 17595510 (July 1st, 2023).
    This formula is based on the following whitepaper; Btc+Eth+Dogecoin+DogeP multi-fork blockchain.
  2. 2) The ERC20 DogeP tokens can be converted to community tokens in the DP blockchain at some point of time after mainnet. This is for community initiatives such as GameChain. This is not dependent on the snapshot taken on July 1st 2023, but rather will be based on the number of ERC20 DogeP tokens in the wallet address at the time of conversion after mainnet. The ratio of DogeP tokens to community tokens is 1:1.
    Though the list below does not show the addresses that got tokens after July 1st, 2023, the new addresses that got tokens after July 1st, 2023 can also convert their DogeP tokens to community tokens.
  3. 3) There will be a cut-off time before which both the above conversions need to be done (to be specified around mainnet time), depending on how soon quantum computers are expected to break classical cryptography.
  4. 4) The mainnet snapshot data is also available as a raw csv file.
  5. 5) Some of the data has been rounded for readability.
